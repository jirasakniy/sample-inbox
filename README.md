How to compile and run
1.	Install node.js
2.	Run command “npm install” in 
{your repository folder}\SampleInbox\WebApplication1\scripts\react
3.	Run command “npm run webpack” in 
{your repository folder}\SampleInbox\WebApplication1\scripts\react
4.	Run command “npm start” in 
{your repository folder}\SampleInbox\WebApplication1\scripts\react

5. http://localhost:8080/
